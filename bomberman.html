Bomberman

About the Project

Bomberman is a classic video game where players control a character who must navigate through a maze while placing bombs to defeat enemies and destroy obstacles. The game offers both single-player and multiplayer modes, providing hours of fun and excitement.

Features

  • Classic gameplay with modern enhancements
  • Single-player and multiplayer modes
  • Dynamic maze generation
  • Power-ups and bonuses

Technology Stack

The game is developed in C++ and is compatible with both Linux and Windows operating systems. It utilizes various libraries and frameworks for graphics rendering, audio, and input handling.

Platforms:

  • Linux
  • Windows

Libraries/Frameworks:

  • SDL (Simple DirectMedia Layer)
  • OpenGL (for graphics)
  • OpenAL (for audio)
