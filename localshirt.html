Localshirt

About the Project

The LocalShirt project aims to promote local businesses by providing a platform for them to sell custom-designed shirts. We believe in supporting local communities and empowering small businesses.

Features

  • Custom shirt designs
  • Local business promotion
  • Easy ordering process
  • Secure payment options

Technology Stack

The project is built using HTML, CSS, and JavaScript for the frontend, and Python with Django framework for the backend.
